# 🇱🇰 Sri Lanka Economy Pulse Dashboard (2010–2024)

## 📊 Project Overview

This project analyzes Sri Lanka’s economic performance using key indicators such as GDP growth, inflation, exchange rates, and trade balance.

The goal is to transform raw economic data into meaningful insights using data analytics and visualization techniques.

---

## 🔍 Data Sources

* Central Bank of Sri Lanka
* World Bank
* Department of Census and Statistics Sri Lanka

---

## ⚙️ Process

1. Data collection from multiple sources (CSV & PDF)
2. Data scraping and preprocessing using Python
3. Data cleaning and transformation
4. Dashboard development using Power BI

---

## 📈 Key Insights

* GDP growth trends across years
* Inflation fluctuations and patterns
* Exchange rate (LKR vs USD) behavior
* Import vs export contribution to GDP

---

## 🛠 Tools & Technologies

* Python (Pandas, Data Cleaning)
* Power BI (DAX, Visualization)
* Data Analysis
